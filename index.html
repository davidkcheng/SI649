• Circles: web pages.
    Links: hyperlinks.

    The circle size encode the hub value
    The darness of color encodes the authority value

  • This is the interactive visualization of Hyperlink-Induced Topic Search (HITS). Choose the number of nodes or probability of links (higher probability means probably more links!) to generate a new graph.

    You may also edit the current graph:
    Click a node/link + delete: delete node/link
    Click anywhere: create new node
    Drag a line from node to node: create new link
    Note: editting graph is not available under convergence exploration mode
